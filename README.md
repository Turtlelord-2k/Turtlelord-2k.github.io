## 🤖 About

This is the portfolio website of **Bhushan Ravikumar**, a Systems Engineer specializing in **Robotics** and **Medical Device Development**. The website showcases expertise in systems engineering, R&D, and innovative technology solutions with a modern, animated interface inspired by robotics and medical technology themes.

## 🚀 Features

### 🎨 Design & Animation
- **Futuristic Theme**: Robotics and medical device inspired design
- **Smooth Animations**: CSS3 and JavaScript powered animations
- **Interactive Elements**: Hover effects, particle systems, and dynamic content
- **Responsive Design**: Optimized for all devices and screen sizes

### 🛠️ Technical Features
- **Progressive Web App (PWA)**: Installable with offline functionality
- **Performance Optimized**: Fast loading with efficient animations
- **SEO Optimized**: Meta tags and structured data
- **Accessibility**: WCAG compliant with keyboard navigation
- **Modern JavaScript**: ES6+ features with fallbacks

### 📱 Interactive Components
- **Animated Robot**: Interactive eyes that follow mouse cursor
- **Typing Animation**: Dynamic role titles in hero section
- **Skill Bars**: Animated progress indicators
- **Timeline**: Interactive experience timeline
- **Particle System**: Floating background animations
- **Medical Device Simulator**: Animated heartbeat and LED indicators

## 🏗️ Built With

### Frontend Technologies
- **HTML5**: Semantic markup and accessibility features
- **CSS3**: Custom properties, Grid, Flexbox, and animations
- **Vanilla JavaScript**: Modern ES6+ without frameworks
- **Progressive Web App**: Service worker and manifest

### External Libraries
- **[AOS (Animate On Scroll)](https://michalsnik.github.io/aos/)**: Scroll-triggered animations
- **[Font Awesome](https://fontawesome.com/)**: Icons and symbols
- **[Google Fonts](https://fonts.google.com/)**: Orbitron and Exo 2 typography

### Design Elements
- **Color Palette**: Cyan (#00d4ff), Teal (#4ecdc4), Medical Green (#00ff88)
- **Typography**: Orbitron (headings), Exo 2 (body text)
- **Animations**: Custom CSS keyframes and JavaScript interactions

## 🚀 Deployment

### GitHub Pages Setup

1. **Fork or Clone** this repository
2. **Enable GitHub Pages** in repository settings
3. **Select Source**: Deploy from `main` branch
4. **Custom Domain** (optional): Configure in repository settings

### Local Development

```bash
# Clone the repository
git clone https://github.com/Turtlelord-2k/Turtlelord-2k.github.io.git

# Navigate to project directory
cd Turtlelord-2k.github.io

# Serve locally (Python 3)
python -m http.server 8000

# Or with Node.js
npx serve .

# Open in browser
open http://localhost:8000
```

## 🎨 Customization

### Color Scheme
Modify CSS custom properties in `style.css`:

```css
:root {
    --primary-color: #00d4ff;    /* Main accent color */
    --secondary-color: #ff6b6b;  /* Secondary accent */
    --accent-color: #4ecdc4;     /* Tertiary accent */
    --medical-green: #00ff88;    /* Medical theme color */
}
```
